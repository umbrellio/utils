# Umbrellio Utils

## Installation

Add this line to your application's Gemfile:

```ruby
gem "umbrellio-utils"
```

And then execute:
```bash
$ bundle install
```
Or install it yourself as:
```bash
$ gem install umbrellio-utils
```
